# 星萌远征 - 文字挂机刷图页游原型

这是一个纯静态的文字挂机刷图页游原型，可直接打开 `index.html` 试玩。

## 本地运行

直接双击 `index.html`，或在本目录启动静态服务：

```bash
python3 -m http.server 8080
```

然后访问 `http://127.0.0.1:8080`。

## 已实现玩法

- 独立登录/注册页，本地账号和游戏角色分开
- 注册/登录后创建训练师：填写游戏名字并选择初始伙伴
- 自动恢复最近账号或角色
- 72 个原创伙伴，包含 N/R/SR/SSR/UR 稀有度、元素、定位和技能文本
- UR 伙伴每个元素系保留 1 只，图鉴按稀有度排列
- 5 人阵容，上阵伙伴根据元素和定位获得协同加成
- 闯关界面整合 5 章 40 关主线和试炼塔，支持手动挑战和自动闯关
- 元素相克影响主线和试炼塔预计胜率，闯关记录会保留最近结果
- 离线挂机收益，最多累计 8 小时
- 快速扫荡、手动挑战下一关、切换挂机地点
- 宠物抽奖分金币池和钻石池：金币池每日 10 次，钻石池产出更高品质伙伴
- 30 件原创装备，装备品质按 N/R/SR/SSR 划分并支持穿戴
- 重复伙伴可用于强化：N/R 最高 +3，SR 最高 +5，SSR 最高 +10，UR 最高 +20
- 商店可购买防跌卡和幸运卡，强化指南展示各品质各等级成功率
- 试炼塔、任务、邮件补给、商店、战报记录

## 版权边界

该原型没有使用停服游戏的名称、角色、美术、剧情或素材。它参考的是同类挂机 RPG 的通用玩法结构：挂机收益、推图解锁、更高关卡收益、角色抽取、阵容定位、升星和挑战塔。后续可以替换成你自己的世界观、数值、角色和美术。

## 存档说明

试玩账号和游戏进度保存在当前浏览器的 `localStorage`。换电脑、换浏览器或清空浏览器数据后不会同步。正式上线需要接入后端账号、数据库和鉴权。

当前密码仅用于本地原型流程演示，保存在浏览器本地存储中，不适合直接用于正式运营。
